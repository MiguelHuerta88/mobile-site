1) sync over all files in mobile-site using Cyberduck to hosting server at /home/projec64/sync
2) ssh into server and run rsync to copy over file
3) run npm commands and composer on deploy directory
4) copy over all contents in deploy to html_plublic/

    --exclude "deploy.sh" \
    --exclude ".idea/" \
    --exclude ".vagrant/" \

    jenkins@$static_server:/var/www/deploy-int
